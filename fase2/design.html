Etiquetas básicas de HTML, atributos y valores

HTML es un markup language, lo que significa que está escrito con códigos que puede leer una persona sin que sea necesario compilarlo primero
Estas etiquetas de marcado son las que se denomina etiquetas HTML.
Entre las etiquetas basicas encontramos body, head, div, a, strong.
